# 📌 프로젝트명: SMP

> 지역 기반 정보 공유 및 채팅 서비스 (React + Kakao Map API)

## 🔍 소개

**SMP**는 사용자의 위치 기반으로 지역 정보를 공유하고,  
실시간 채팅을 통해 소통할 수 있는 커뮤니티 플랫폼입니다.  
비 오는 날 잠깐 비추는 햇살처럼, 일상 속 따뜻한 연결을 제공합니다.

## 🛠️ 주요 기능

- 📍 **카카오 맵 기반 위치 검색**
- 🧵 **동네 채팅방 생성 및 참여**
- 💬 **실시간 채팅 기능 (socket.io 예정)**
- 📝 **게시글 및 댓글 기능**
- 📱 **반응형 디자인**
